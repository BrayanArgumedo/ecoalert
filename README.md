# EcoAlert 🌿

Sistema ciudadano de reporte de emergencias ambientales.

## Descripción

EcoAlert conecta a ciudadanos afectados por emergencias ambientales con los servicios públicos de respuesta (bomberos, policía, asistencia médica) de manera ágil, priorizada e inteligente.

## Stack

- **Mobile:** React Native + Expo SDK 54 + TypeScript + NativeWind
- **Backend:** Node.js + Express + TypeScript
- **Base de datos:** MySQL 8.0 (Docker)
- **Auth:** JWT + Bcrypt + RBAC
- **API Testing:** Bruno

## Estructura del proyecto
```
ecoalert/
├── backend/          # API REST Node.js + Express
├── mobile/           # App React Native + Expo
├── infrastructure/   # Docker + MySQL
├── bruno-collection/ # Colección de API testing
└── docs/             # Documentación
```
